# STRM

Search any movie or series from your terminal. Type a name. Pick a result. Watch instantly.

![STRM](https://strm-website.vercel.app/screenshot.png)

## Install

```bash
curl -fsSL https://strm.sh/install.sh | bash
```

## Usage

```bash
strm
```

- Type a movie or series name
- Browse results with arrow keys
- Press Enter to watch
- Press Esc to go back
- Press `Ctrl+C` to quit

## How it works

STRM searches [TMDB](https://www.themoviedb.org/) for movies and series, then opens the player in your browser. The player supports season/episode selection for TV shows.
